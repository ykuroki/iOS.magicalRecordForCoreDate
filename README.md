magicalRecordForCoreDate
==============

* Core Dateを簡略化してくれるMagical Recordがかなり便利ということ使ってみました
* 対応OS: iOS 7.1 or later

ビルド方法
==============

 * Xcode 5
 * [CocoaPods](http://cocoapods.org) を使用しています。

```sh
// プロジェクトルートで
pod install
open ./coredata.xcworkspace
```
